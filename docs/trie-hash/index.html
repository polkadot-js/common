trie-hash

@polkadot/trie-hash

Calculate ordered and unordered Ethereum Trie hashes from inputs.

Usage

Installation -

yarn add @polkadot/trie-hash

Functions can be imported as follows:

import { trieRoot } from '@polkadot/trie-hash';
